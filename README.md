## FaS v0.9 ##
First of all "FaS" means "Fast As a Shark". It is a shell script that makes it possible to configure linux servers quickly as possible. It has an interface and easy to use.

Usage :
``` 	 	
$vi fas.sh #Paste script

$chmod +x fas.sh

$./fas.sh
```
